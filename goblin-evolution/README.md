# Эволюция Гоблина

HTML5-кликер для Яндекс.Игр. Играть локально:

```bash
python3 -m http.server 8000
# откройте http://localhost:8000/goblin-evolution/
```

## Содержимое

- `index.html` — разметка и подключение SDK Яндекс.Игр.
- `styles.css` — стили, фоны по уровням, анимации и UI.
- `game.js` — логика игры: клик, экономика, апгрейды, уровни, сейв в `localStorage`.
- `sdk.js` — обёртка над `YaGames` с методами `showFullscreen` и `showRewardedVideo`.

## Загрузка в Яндекс.Игры

1. Заархивируйте содержимое папки `goblin-evolution/` в ZIP (не саму папку, а её файлы).
2. Загрузите ZIP в консоль разработчика Яндекс.Игр.
3. При необходимости подключите rewarded-видео к бонусу в игре через `YSDK.showRewardedVideo`.

## Реклама

- Межстраничная (`showFullscreenAdv`) вызывается на повышении уровня гоблина, но не чаще раза в 180 секунд.
- Rewarded-видео (`showRewardedVideo`) готово к использованию, но пока не привязано ни к одному бонусу — подключается одним вызовом в `game.js`.
